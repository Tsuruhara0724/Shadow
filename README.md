# Shadow Body

**ユング心理学の「影の理論」をテーマにした2人協力オンラインパズルアクションゲーム**

▶ [プレイ動画](https://youtu.be/pgeK2l5-TIM?si=FqZGvIZ8NbwZ3MOW)

---

## ゲーム概要

プレイヤー1は太陽を操作して影の形を変え、  
プレイヤー2はその影の中でのみ移動できる「影の球」を操作する。  
2人がリアルタイムで連携し、各ステージをクリアしていく。

ゲームのテーマ・ステージ名・ビジュアルスタイルはすべて  
ユング著『赤の書（Liber Novus）』から着想を得ています。

※ ゲーム内テキストは中国語です。ご了承ください。

---

## 操作方法

| プレイヤー | 操作 |
|---|---|
| 太陽（P1） | W/S=上下　A/D=左右　Q/E=前後（※太陽の高度・角度を変えて影の形状を操作） |
| 影の球（P2） | テンキー方向キー=移動　スペース=ジャンプ |

LAN接続：同一LAN内でホスト側が「Host」ボタンを押し、
クライアント側がIPアドレスを入力して「Client」で参加します。
接続後、BキーとGキーでキャラクターを選択します。

共通
Esc キー → タイトル画面に戻る　／　空気壁に触れると現在ステージの起点にリスポーン

## 技術情報

- **エンジン** : Unity 2022.3.20f1c1
- **言語** : C#
- **ネットワーク** : Mirror + KCP Transport（LAN通信）
- **シェーダー** : Amplify Shader Editor（ASE）
- **設計パターン** : Singleton / Observer
- **開発期間** : 約1.5か月（個人制作）
